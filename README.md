# 环境变量

| 变量名      | 信息               |
| ----------- | ------------------ |
| COOKIE`*`   | HiFiNi的cookie信息 |
| SERVER_CHAN | Service酱推送的key |

`*`:表示必选。



## 获取HifiNiCookie
首页`F12`打开调试工具,在请求表头中找到并复制cookie的值
![image](https://github.com/anduinnn/HifiNiAutoCheckIn/assets/68073009/97528823-4d31-4c72-bcca-e95bb5d75792)
